### Description
This project is about function that return a line read from file descriptor. It has main function which is in get_next_line.c and sub function which is in get_next_line_utils.c.

### Instruction
To compile the project (a buffer size of 42 is used as an example):  
cc -Wall -Wextra -Werror -D BUFFER_SIZE=42 <files>.c
